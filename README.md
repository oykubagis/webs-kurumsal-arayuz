WEBS - Kurumsal Web Arayüzü Tasarımı 

  Bu proje, modern web tasarım prensipleri ve Bootstrap framework'ü kullanılarak geliştirilmiş çok sayfalı bir kurumsal web sitesi ön yüz (frontend) çalışmasıdır. 

Proje Hakkında
   Bu çalışma, bir işletmenin ihtiyaç duyabileceği ana sayfa, hakkımızda (ekibimiz/biyografi) ve kullanıcı kayıt/giriş sayfalarını içermektedir. Tasarımda kullanıcı deneyimi (UX) ve responsive (mobil uyumlu) yapı ön planda tutulmuştur.

Özellikler
- Responsive Tasarım: Mobil, tablet ve masaüstü cihazlarla tam uyumlu grid yapısı.
- Dinamik Bileşenler: Bootstrap Carousel (kaydırıcı), hover efektli kartlar ve modal yapısı.
- Modern UI: CSS3 animasyonları ve özelleştirilmiş stil dosyaları.
- Çoklu Sayfa Yapısı: Ana sayfa, Hakkımızda ve Kayıt Ol sayfaları arasında tutarlı navigasyon.

Kullanılan Teknolojiler
- HTML5
- CSS3 
- Bootstrap 4.6
- Font Awesome 6
